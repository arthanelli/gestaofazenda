----GIT-----
Se você ainda não tem o repositorio clone ele para sua maquina com o comando: git clone url
Caso voce tenha o repositorio não rode esse comando.

Para subir para o repositorio suas mudanças voce tem que colocar elas em um commit que você ira criar.
Veja os arquivos que tiveram mudanças com o comando: git status
Todos os arquivos em vermelho serão adicionadas ao commit rodando esse comando: git add .
logo após coloque um nome no commit: git commit -m "o que voce fez"
Depois disso voce já guardou suas mudanças, agora suba no repositorio para ficar disponivel para todos: git push origin master

Para atualizar seu projeto com o que está na nuvem: git pull origin master
Sempre que começar a programar rode esse comando.

----PROJETO----
Você precisa ter o node instalado na maquina.
E o git bash.

Primeiro comando é o npm install para instalar todas as dependecias do projeto.
logo após instale o modulo nodemon para verificar as mudanças do codigo e atualizar o servidor, comando: npm install -g nodemon
para rodar o projeto rode o comando: npm start
